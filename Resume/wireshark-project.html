Project: HTTPS Traffic Analysis Using Wireshark

In this project, I captured and analyzed network traffic using Wireshark while accessing multiple websites
over HTTPS. I applied advanced display filters to isolate TLS handshake packets and identify secure web traffic
on TCP port 443. Conditional logic was used to include and exclude traffic from specific IP addresses, ensuring
accurate filtering results. Parentheses were implemented in compound filters to control execution order and
prevent logical errors. The project demonstrated how encrypted web traffic can be identified, analyzed, and
narrowed down within a packet capture without decrypting payload data.

Skills Learned:

  • Wireshark Display Filtering
  • Understanding Secure Web Communication
  • Troubleshooting and Analytical Thinking
